===============
Getting Started
===============

At the command line:

.. code-block:: bash

    git clone https://github.com/URVPenguin/minecraft_agent_framework.git
    cd minecraft_agent_framework
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
